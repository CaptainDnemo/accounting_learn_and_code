会计基础

导论

  1. 会计的定义：商业语言的表达
  2. 会计的目的：记录价值的交换

会计对象与会计假设

  1. 三个要素：主体，时点，金额
  2. 识别主体：
    • 法律主体和会计主体一致
    • 会计主体超越法律主体
    • 首先要明确会计主体
  3. 其他假设：
    • 主体：持续经营假设
    • 时点：会计分期假设
      1. 公历年度：xxxx年1月1日-xxxx年12月31日
      2. 自然营业年度：a年b月1日-a+1年b-1月31日
    • 金额：货币计量假设
  4. 谨慎性原则
    • 应选择低估好处、高估损失的方法

会计要素与会计恒等式

水流不腐，人活不输

  1. 单式记账：只记录资金流入流出
    • 问题：
      1. 业务复杂的情况
      2. 债权增多
      3. 股东投入
      4. 整体盈利情况
      5. 纠错能力不足
  2. 复式记账：记录资金的去向和来源

    有借必有贷，借贷必相等 - 好处： 1. 区分不同种类的来源和去向 2. 易于检查、避免错误

  3. 会计恒等式：
    • 目的：
      1. 在一个时点上，只能看到资金运动的结果和来源
      2. 在一段时间内，能看到资金运动变化的情况
    • 等式：
      1. 资产(借)=负债+所有者权益(贷)
      2. 收入-费用=利润
    • 举例：
      1. 股东认缴注册资本x元：
        -> 银行存款(资产)+x
        -> 实收资本(所有者权益)+x
      2. 企业从银行存款中提取x元现金备用：
        -> 现金(资产)+x
        -> 银行存款(资产)-x
      3. 企业从供应商得到面粉等原料，总价x元，钱未付清：
        -> 存货(资产)+x
        -> 应付账款(负债)+x
      4. 企业购买设备，用银行存款支付x元：
        -> 固定资产(资产)+x
        -> 银行存款(资产)-x
      5. 偿还债务x元：
        -> 银行存款(资产)-x
        -> 应付账款(负债)-x
  4. 资产：
    • 性质：
      1. 能带来以货币计量的利益
      2. 企业控制或拥有
    • 分类：
      1. 资金资产/非现金资产
      2. 流动资产/非流动资产
      3. 隐形资产
    • 流动资产：短期内能变成现金
      1. 库存现金/银行存款
      2. 应收票据：客户出具的商业汇票
      3. 应收账款：客户口头承诺没有票据
      4. 预付账款：预付给供应商的贷款
      5. 存货：在一个营业周期内
    • 非流动资产：短期内不能变成现金
    • 隐形资产：没有记录到会计系统内
      1. 品牌：无法用货币可靠计量
      2. 人力资源：不是企业拥有的
      3. 关系：以上两个问题都有
      4. 商业合同：尚未履行
  5. 负债：
    • 性质：
      1. 过去事项->当下亏欠
      2. 履行义务->未来一个时点经济利益流出
    • 分类：
      1. 流动负债/非流动负债
    • 流动负债：(和流动资产相反)
      1. 短期借款
      2. 应付与预收款项
      3. 应交税金
    • 长期负债：
      1. 长期借款
      2. 应付债券
      3. 长期应付款
    • 隐形负债：
      1. 法律诉讼
      2. 担保
      3. 其他(退货承诺)
  6. 所有者权益：
    企业投资者对企业的剩余索取权
